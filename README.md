# 🚀 Taskflow – Advanced To-Do App

Taskflow is a modern, full-featured To-Do web application built using HTML, CSS, and JavaScript. It provides a clean UI and powerful task management features to help users stay organized and productive.

---

## ✨ Features

* 📝 Add, edit, and delete tasks
* ✅ Mark tasks as completed
* 📂 Task categories (Work, Personal, Study, etc.)
* 🎯 Priority levels (Low, Medium, High)
* 📅 Due date tracking with overdue alerts
* 🔍 Search tasks instantly
* 🔽 Filter tasks (All / Pending / Completed)
* 📊 Sort by date or priority
* 📈 Progress tracker (circular progress bar)
* 🌗 Dark / Light mode toggle
* 💾 Data saved using LocalStorage
* 📱 Fully responsive design
* 🖱️ Drag and drop to reorder tasks
